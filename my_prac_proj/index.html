How dark mode works:

When you click the "Toggle Dark Mode" button, a JavaScript function uses classList.toggle('dark-mode') on the <body>. This adds the dark-mode class if it's not present, or removes it if it is. CSS (in style.css) can then style the page differently for dark mode.
